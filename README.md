# homebrew-pomodoro

Homebrew tap for Qoma.

## Install

```sh
brew tap dzenn-app/pomodoro
brew install --cask qoma
```

## Update

```sh
brew update
brew upgrade --cask qoma
```

## Notes

This tap is for direct downloads from GitHub Releases.
If macOS blocks the app (unsigned), use:
System Settings -> Privacy & Security -> Open Anyway.
